| Rayo Capital         | rayo-capital.png           | dark navy              |

## Notes
- Recommended dimensions: ~320×120px or similar landscape format
- PNG with transparency works best
- If a logo file is missing, the card automatically falls back to the partner's styled name text
- To update the background colour of a card, edit the `style` property in `src/components/Partners.tsx`
